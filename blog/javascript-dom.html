Explain control flow and loops using an example process from everyday life

Control flows is the order in which instructions in a program are executed. Loops are a type of control flow that allow a program to repeat a set of instructions until a certain condition is met. An example of a control flow is like making a bowl of cereal 1. Open the cereal box and remove the bag of cereal 2. Pour the desired amount of cereal into a bowl 3. Check the milk level in the container 4. If there is enough milk, pour milk over the cereal until it is covered 5. If there is not enough milk, refill the milk container 6. Add sugar (if desired) 7. Use a spoon to mix the cereal, milk, and sugar together 8. Enjoy the cereal Loops are not nessecary in this process but we can use a loop to make multiple bowls if we want. all we would need to do is repeat steps 2-7 until we had the desired amount of bowls. Overall Control flows are the order of a process and loops are the repeats we can put into the process to get meet conditions we would like to achieve.

Describe what the DOM is and an example of how you might interact with it.

The DOM stand for Document Object Model and it provides the means to interact with objects in an HTML document. The DOM creates a tree-like structure of all the elements in a webpage. Each element is called a node and each node has properties that can be accessed by developers to make changes through Javascript

You can use the DOM to change the colour of a button when someone clicks on it or to add a new picture or change text based on the users input. its a powerful tool to make websites come alive.
